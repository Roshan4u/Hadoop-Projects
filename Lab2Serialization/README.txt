This project is a practive on customerizing Writable Objects in Hadoop, and implement a write and read script for sequence file. 
* TextPair: Writable<Text,Text>
* TextIntPair: Writable<Text,Text,IntWritable>
* NameSprintCount: count sum of values on key<name, sprint>
* WriteSequence: arguments(input path, output path)
* ReadSequence: argument(sequence file path), display normal context in console.
